📚 Python Data Structures Toolkit (Freshman Year Archive)
This repository serves as a curated archive of fundamental data structures I implemented during my 1st year (2025) of Computer Engineering.

🎯 Project Overview
The core objective of this project was to master the low-level logic of data management by building essential structures from scratch. As a current 2nd-year student, I maintain this repository to showcase my foundational journey and logical progression in software engineering.

🛠 Features & Implementations
This toolkit includes pure Python implementations of:

Stack: LIFO (Last-In, First-Out) operations.

Queue: FIFO (First-In, First-Out) operations with basic enqueue/dequeue logic.

Deque: Double-ended queue for flexible data handling.

Linked List: Node-based pointer management and traversal.

💡 Why This Matters
Understanding these structures was my first step toward efficient algorithmic thinking. In the context of my career goal in Cybersecurity and SOC Analysis, these fundamentals are critical for:

Analyzing how data moves through memory.

Understanding network packet buffering.

Efficient log processing and event sequencing.
